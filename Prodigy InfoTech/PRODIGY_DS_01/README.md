# 📊 Task 01 – Data Visualization (Prodigy InfoTech Internship)

## 🚀 Objective

The objective of this task is to create a **bar chart or histogram** to visualize the distribution of a categorical or continuous variable.

---

## 📁 Dataset

* Dataset used: **World Bank Population Dataset**
* It contains population data of different countries across multiple years.

---

## 🧹 Data Preprocessing

* Removed unnecessary metadata rows using `skiprows`
* Handled missing values using `.dropna()`
* Selected relevant column (**Population in 2020**) for analysis

---

## 📊 Visualizations

### 1️⃣ Histogram (Continuous Variable)

* Plotted a histogram to understand the **distribution of population across countries**
* Helped identify how population values are spread

### 2️⃣ Bar Chart (Categorical Representation)

* Created a bar chart for **Top 10 most populated countries**
* Compared population values across countries

---

## 🛠️ Tools & Libraries Used

* Python 🐍
* Pandas
* Matplotlib

---

## 📌 Key Learnings

* Data cleaning is an important first step before visualization
* Histograms are useful for understanding distribution
* Bar charts help in comparing categorical data
* Visualization makes data easier to interpret
